A litlle web browser to navigate through the web

- [Specs](#specs)

- [Preview](#preview)

## Specs

- [XCode 13.4.1](https://xcodereleases.com/)
- Swift 5.6.1

## Installation

- You need to have [CocoaPods](https://cocoapods.org) installed
- Install dependencies 
  ```dependencies
    pod install
  ```
- Run the project in XCode
